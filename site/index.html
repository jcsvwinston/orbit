orbit
mission control
for your app.

A deep, pluggable admin for the Nucleus Go framework — Data Studio, live request/SQL telemetry, sessions, RBAC and metrics. Mounted in-process, served from an embedded UI.

// the console

Everything an operator needs, in one panel.

Orbit reads the running app's Runtime — it sees live state an out-of-process tool never could.

01

Data Studio

Browse, create, edit and delete records for every registered model. Tenant-aware, with import/export.

02

Live inspector

Real-time feed of incoming HTTP requests and executed SQL across the whole app — with optional cross-node aggregation.

03

Session viewer

List and revoke active server-side sessions, live.

04

Access control

Inspect and manage the Casbin policies and roles backing the app's authorizer.

05

System metrics

Runtime and resource consumption — CPU, memory, goroutines, the database pool.

06

Audit & overview

An in-memory audit ring of admin actions, plus a dashboard and health at a glance.

// liftoff

One dependency. One mount call.

No sidecar, no separate asset deployment, no database of its own.

STEP 01

Add the module

go get github.com/jcsvwinston/orbit@latest

STEP 02

Mount it

One .Mount(orbit.Module(…)) on the builder — it acquires what it needs from the Runtime.

STEP 03

Open /admin

Sign in with the bootstrap user. Orbit self-gates its own auth and serves its embedded UI.

 main.go
// mount the admin on the application builder
app, err := nucleus.New().
    FromConfigFile("nucleus.yml").
    Mount(orbit.Module(orbit.Config{
        Prefix:            "/admin",
        Title:             "Acme Admin",
        BootstrapUsername: "admin",
        BootstrapEmail:    "admin@acme.test",
    })).
    Build()
